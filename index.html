nya-Bootstrap-Select

An AngularJS directive wrapper for silviomoreto's Bootstrap-select, which supports ngRepeat in option tag and ngOptions to dynamically build a Bootstrap-select

Getting started

nya-Bootstrap-Select is a wrapper for Bootstrap-select. you can use most of the Bootstrap-select features combine with angular data-binding.

Dependencies

AngularJs 1.0+, jQuery 1.7+, Bootstrap-select 1.3+, Bootstrap css files

Both of Bootstrap 2.x and 3.x are supported.

Installation

You can use bower to install the latest version of this directive.

bower install nya-bootstrap-select

Bower will download all dependencies for you. You need reference these scripts

<script src="bower_components/jquery/jquery.js"></script>
<script src="bower_components/angular/angular.js"></script>
<script src="bower_components/bootstrap-select/bootstrap-select.js"></script>
<script src="bower_components/nya-bootstrap-select/src/nya-bootstrap-select.js"></script>

Add bootstrap-select styles and bootstrap.css. Both bootsrap 2 and 3 are supported.

<link rel="stylesheet" href="bower_components/bootstrap-select.css"/>

Add dependencies of you angular app

angular.module('yourApp', ['nya.bootstrap.select']);

Now, you can use nya-selectpicker attribute or class in select tag with ng-model directive. For varieties usage. See the examples below

Static options

The basic usage of this directive is the static options. retrieve the value of select from ng-model

This example demonstrates a basic usage of the directive. use three hard coded <option> tag in <select> tag. Then add nya-selectpicker class to <select> tag. Add a ng-model directive to retrieve you select value. add data-container="body" to fix the overflow issue.

Dynamic options

Dynamic options using ng-repeat directive give you fully controllable of the options and flexibility of the data. You can use this feature to implements most of the Bootstrap-select features with variable data.

In this example we use ng-repeat directive in <option> tag the generate multi options from data. Click change options will randomly change the data.

Dynamic model

This feature implements the dual-direction bind of ng-model. Give you ability to change selection of bootstrap-select from you program. For example, you can get data via ajax and bind data to model to change the default selection of a select

ngOptions example

Sometimes, using ng-repeat is not necessary and complicated. Instead using ng-repeat to generate a set of option, you can use ng-options directive to generate the options.

ngOptions (Group)

if you want group your options. Using ng-options instead of ng-repeat is usually a better choice. As ng-options syntax and this directive limits. You should organize you data using an array of objects which is like this format: [{value: 'alpha', group: 'group1'}, {value: 'beta', 'group'},...]. Then you can using the following expression to generate options: ng-options="c.value group by c.group for c in myOptions" this expression will populate option value using 'value' property specified by c.value. And populate optgroup using 'group' specified by group by c.group.

Note that if your using ng-options. the model is an object or an array of object (when multiple selection is enabled).
